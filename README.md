# What Will I Be? Website

This folder is ready for GitHub Pages.

## Files

- `index.html` - the website homepage and prototype experience.

## Publish On GitHub Pages

1. Create a GitHub repository named `what-will-i-be`.
2. Upload `index.html` from this folder.
3. Open repository **Settings**.
4. Go to **Pages**.
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Click **Save**.
7. Wait a few minutes for the website link.

Your first link will look like:

`https://YOUR-GITHUB-USERNAME.github.io/what-will-i-be/`

Later, connect your Namecheap domain to this GitHub Pages site.
