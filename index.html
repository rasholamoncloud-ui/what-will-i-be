Quick Chat

Prototype view for real-time support between learners, parents, teachers, and moderators.

Teacher: Which career family should I use for a 10-minute assembly talk?
Moderator: Start with one family, show two flashcards, sing the short song, then ask one observation question.
Parent: My child likes drawing clothes. Where should I begin?
Moderator: Try Fashion & Personal Design, then compare it with Visual & Design Arts.

Launch requirement: this needs user accounts, moderation, abuse reporting, logging, and protected real-time infrastructure.
